## Instalación

1. Git clone:
    ```
         git clone https://github.com/svaltqt/OPConConjuntos.git
    ```
2. Instalacion de JavaFX (Dentro del IDE)
    ``` 
         Se realiza desde el Maven instalando dependencias desde el ícono 'M'
    ```

## Uso

Abrir la consola y usar el comando
 ``` 
     mvn javafx:run
 ```
